# Prototipo de Automatización de Procesos
## Proyecto Final – Modelamiento y Automatización de Procesos
Ingeniería de Software – UPB
## Descripción del proyecto
Este repositorio contiene un prototipo funcional que simula la automatización de un proceso de despliegue de software. El prototipo fue desarrollado como parte del proyecto final del curso **Modelamiento y Automatización de Procesos**.

El objetivo del proyecto es demostrar cómo un proceso manual (AS-IS) puede ser rediseñado (TO-BE) e implementado parcialmente mediante automatización utilizando un pipeline CI/CD.
El sistema simula la ejecución automática de las siguientes etapas del proceso:
* Inicio del pipeline
* Proceso de build
* Ejecución de pruebas
* Despliegue del sistema
* Generación de artefactos
* Notificación automática
Este prototipo representa de forma visual el proceso automatizado definido en el modelo TO-BE del proyecto.

El proyecto contiene los siguientes archivos:

* **script.js** – simulación del pipeline automatizado
* **README.md** – documentación del proyecto

## Funcionamiento del prototipo
El usuario puede ejecutar una simulación del pipeline presionando el botón **“Ejecutar Pipeline”**.
El sistema simula automáticamente las siguientes etapas:
1. Inicio del pipeline
2. Build del proyecto
3. Ejecución de pruebas
4. Despliegue automatizado
5. Generación de artefacto
6. Notificación del resultado
Este flujo representa el proceso automatizado definido en el modelo TO-BE del proyecto.

## Relación con el proyecto académico
El prototipo demuestra de manera conceptual la automatización de un proceso de despliegue de software, integrando los conceptos estudiados durante el curso:
* Modelamiento de procesos (BPMN)
* Mejora de procesos (AS-IS → TO-BE)
* Automatización mediante pipelines CI/CD
* Integración conceptual con RPA
